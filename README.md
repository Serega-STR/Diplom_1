## Задание 1 

**Юнит-тесты**

Автотесты для проверки программы, которая помогает заказать бургер в Stellar Burgers

### Реализованные сценарии

Созданы юнит-тесты, покрывающие класс `Burger`

Процент покрытия 100% (отчет: `htmlcov/index.html`)

### Структура проекта

- `praktikum` - пакет, содержащий код программы
- `tests` - пакет, содержащий тесты.

### Запуск автотестов

**Установка зависимостей**

> `$ pip install -r requirements.txt`

**Запуск автотестов и создание HTML-отчета о покрытии**

>  `$ pytest --cov=praktikum --cov-report=html`

## Просмотр отчета о покрытии

Чтобы просмотреть HTML-отчёт о покрытии кода, выполните следующие шаги:

1. Найдите директорию с отчётом.
По умолчанию pytest-cov создаёт папку htmlcov в корне вашего проекта. Внутри неё будет файл index.html.
2. Откройте htmlcov/index.html в веб-браузере.
